# 自己紹介サイト

## サイトについて
学習目的で制作した自己紹介サイトです。作者の紹介、勉強中の内容、好きなものなどのセクションを掲載しています。

## ビルド
```bash
npm install
npm run build
```

## 公開URL
- https://moppi-self-introduction.netlify.app/

## 設定メモ（Netlify）
- Build command: `npm run build`
- Publish directory: `.`
- 生成されるCSS: `dist/style.css`

## メモ
- `dist/` はビルド成果物のためGit管理しません。
